# AGNEYRA Full-Stack Development Internship

This repository contains projects completed during the **AGNEYRA Full-Stack Development Internship Program**.  
The internship focuses on building practical full-stack applications and improving real-world development skills.

## Internship Details
- **Company:** AGNEYRA
- **Domain:** Full-Stack Development
- **Project ID:** AGY101
- **Duration:** 30 Days Internship

The projects cover backend development, API creation, authentication, database handling, frontend integration, and deployment.

---

## Challenges Completed

### Challenge 1: SkillBridge Authentication System
**Objective:**  
Build a secure authentication system.

**Features:**
- User Signup & Login
- Password Hashing
- JWT Authentication
- Protected Routes using Middleware

**Deliverables:**
- Backend authentication system
- Protected route implementation
- Postman API testing

---

### Challenge 2: Course Management System
**Objective:**  
Allow instructors to create courses and students to view them.

**Features:**
- Course CRUD APIs
- Course listing frontend
- Role-based access (Instructor / Student)

**Deliverables:**
- Working course APIs
- Frontend course listing
- Role-based course creation

---

### Challenge 3: Enrollment & Payment Logic
**Objective:**  
Allow students to enroll in paid courses.

**Features:**
- Enrollment system
- Prevent duplicate enrollments
- Student dashboard showing enrolled courses

**Deliverables:**
- Enrollment API
- Student dashboard
- Database verification

---

### Challenge 4: SkillBridge Deployment & Dashboard
**Objective:**  
Deploy the complete full-stack application.

**Features:**
- Instructor & Student dashboards
- Analytics (students, revenue, courses)
- Frontend & backend deployment
- Environment configuration

**Deliverables:**
- Live deployed platform
- Admin dashboard
- Public demo link

---

## Tech Stack
- Backend: Node.js / Express
- Frontend: HTML, CSS, JavaScript
- Database: MongoDB / SQL
- Authentication: JWT
- Tools: Postman, GitHub
